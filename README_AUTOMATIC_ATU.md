# Future automatic ATU delivery

The current web app uses manual ATU assignment after payment confirmation.

A later version can use a smart contract that:

1. receives BNB or USDC;
2. records membership plan;
3. transfers ATU to the buyer;
4. emits events for accounting and transparency.

Important: do not automate ATU delivery from a normal MetaMask wallet by storing a private key on a website or public server. Use a smart contract that holds a planned ATU allocation, or continue manual assignment.
